Turns the media upload icons into plain text links. 

== Description ==


Fully translated, uses your native translation files.

Send me your bug reports and suggestions via my [contact page](http://toscho.de/kontakt/) or per bugtracker at the [public repository at GitHub](https://github.com/toscho/Icon-to-link).

[Beschreibung auf Deutsch](http://toscho.de/2010/wordpress-plugin-icon-to-link/)


== Installation ==

Upload the directory to your plugin directory. 
Activate the plugin through the 'Plugins' menu in WordPress.
